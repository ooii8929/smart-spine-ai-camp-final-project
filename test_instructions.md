## 常見問題排除

### 如果攝影機無法啟動
1. 檢查瀏覽器網址列是否有攝影機圖示
2. 點擊圖示，確認權限設為「允許」
3. 重新整理頁面

### 如果偵測不準確
1. 確保光線充足
2. 調整與攝影機的距離（50-100cm）
3. 確保上半身完整入鏡
4. 穿著與背景有對比的衣服

### 如果畫面卡頓
1. 關閉其他耗資源的程式
2. 使用 Chrome 瀏覽器（效能最佳）
3. 檢查電腦 CPU 使用率
